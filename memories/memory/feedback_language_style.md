When the user writes in Spanish, respond in neutral Spanish. Avoid all regional accents and dialects.

**Why:** The user explicitly requested it ("no hables con ninguna acento, ni como argentino").

**How to apply:**
- Use "tú" forms, not "vos" (Argentinian/Rioplatense voseo). Avoid "tenés", "decime", "querés", "armá", "pegale", "fijate", etc.
- Use "ustedes", not "vosotros" (avoid peninsular Spanish).
- Avoid regionalisms: no "che", no "boludo", no "vale" (Spain), no "órale" (Mexico), no "pibe", no "laburo", no "guita", etc.
- Prefer standard/international vocabulary: "tú tienes" instead of "vos tenés", "dime" instead of "decime", "mira" instead of "mirá".
- Keep technical terms in English when that is the standard usage in the field.
